# 寿司ガイドが売れていない場合の対策プラン

## 前提
- 出品直後は売上ゼロが普通。フォロワーゼロからのスタート。
- 最初の1週間で重要なのは「閲覧数（views）」。売上はその後。
- views > 0 なら集客は機能している。コンバージョン率の問題。
- views = 0 なら集客自体ができていない。

---

## パターン1: views = 0（誰も見ていない）

### 原因: 集客チャネルが足りない
### 対策:
1. Reddit投稿が削除されていないか確認
2. Quora回答を5問投稿（各質問に数千view）
3. Medium記事を公開（SEOで長期流入）
4. Facebook Japan Travel系グループに参加→投稿
5. Pinterest にピン3つ作成
6. Twitter/Xで寿司知識投稿→プロフィールにリンク

### 目標: 1週間で100 views

---

## パターン2: views > 50 だが売上 = 0（見てるが買わない）

### 原因: 商品ページのコンバージョンが低い
### 対策:
1. **価格を下げる**: $9.99 → $4.99（お試し価格）
2. **無料サンプルを追加**: 最初の3セクションを無料PDFで配布→フルバージョンへ誘導
3. **商品説明を改善**: ベネフィットをもっと具体的に
4. **レビューを獲得**: 知人に無料で配って★5レビューをもらう
5. **表紙画像を改善**: よりプロフェッショナルなデザインに

### 目標: コンバージョン率2-5%（50viewsで1-2売上）

---

## パターン3: 売上1-5個（動いてはいるが30,000円に足りない）

### 原因: 集客量が足りない
### 対策:
1. Medium記事を週2本追加（SEO流入の拡大）
2. Quora回答を追加10問
3. Reddit r/japanlife, r/japan, r/japantravel に追加投稿
4. 価格を$14.99に上げる（少量でも単価で稼ぐ）
5. 「バンドル商品」を作る（寿司ガイド + 日本旅行フレーズ集 = $19.99）

---

## パターン4: 集客はできているが、技術記事ルートの方が早く3万円に届きそう

### 判断基準:
- DigitalOcean/Draft.devから承認メールが来た → そちらに集中
- 両方不承認 → 寿司ガイドの集客を全力で強化

---

## 時間軸

| 日付 | チェックポイント | 判断 |
|------|----------------|------|
| 4/3 | views確認 | 0なら集客チャネル追加 |

| 4/10 | DO/Draft.dev承認確認 | 承認あり → 記事執筆に集中 |
| 4/14 | 中間レビュー | 全ルートの進捗を棚卸し |
| 4/20 | 最終判断 | 3万円未達見込み → セルフバック等のフォールバック |
| 4/30 | 最終確認 | 結果集計 |
